# Mondrian Project
A recreation of a Piet Mondrian-style abstract artwork using HTML and CSS Grid.

## Features
- A grid-based design mimicking Mondrian's iconic abstract style.
- Uses CSS Grid properties for precise layout and placement.

## Technologies Used
- HTML
- CSS (Display Grid)
